You need server to run.
You can create simple python server with command on root folder. i.e. outside of executable folder.
You can run "python -m http.server 8000" command and open "http://localhost:8000/executable/main.html".

=====================================================================================================
You can create toroid and hyperboloid.
You can select a created shape from shape list and change it's properties.
Light properties applied to all objects. Other properties applied to each object.
You can move camera with mouse click and drag.
You can show (or not show) texture or environment maps by selecting.
By default objects have no texture. You need to select texture of the object to see texture mapping.
Environment mapping applied to all objects.
